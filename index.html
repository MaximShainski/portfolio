How I Work

As a student of UI design, I am currently diving into a UI design course. This program is equipping me with foundational knowledge and practical skills in various principles and elements of design. Some design elements I've learned to focus on are:

  • Visual Communication elements such as Colours, Balance, Contrast, Typography, Photography, Iconography, Scale, Visual Hierarchy, etc.
  • User Centered Design principles such as Empathy, User focus, user involvement/testing, iterative designing, etc.
  • Prototyping principles such as paper prototypes, wireframes and in general constantly surveying with users to understand their needs
